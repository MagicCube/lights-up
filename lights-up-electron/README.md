# lights-up-electron

## About

The desktop controller of Lights-up.

## How to Run in Development Mode

1. Install npm packages.
  ```sh
  yarn
  ```

2. Start development web server.
  ```sh
  yarn dev:web 
  ```

3. Start watch mode for electron app.
  ```sh
  yarn dev:electron
  ```

4. Start electron app.
  ```sh
  yarn start
  ```
